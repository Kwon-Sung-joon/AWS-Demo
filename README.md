# Cloud AWS 프로젝트

### (목적)
* AWS의 EC2 인스턴스를 콘솔을 사용하지 않고 프로그램으로 자동 처리 

### 구현범위
EC2 인스턴스 시작 및 종료만 구현 (인스턴스 ID 필요)
